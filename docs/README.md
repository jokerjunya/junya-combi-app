## 📖 読む順序（推奨）

初めてセットアップする場合：

1. [QUICKSTART.md](./QUICKSTART.md) - 全体の流れを把握
2. [FIREBASE_CLI_SETUP.md](./FIREBASE_CLI_SETUP.md) - Firebase設定
3. [SETUP_COMPLETE.md](./SETUP_COMPLETE.md) - 動作確認

開発する場合：

1. [ARCHITECTURE.md](./ARCHITECTURE.md) - アーキテクチャを理解
2. メインの [README.md](../README.md) - API仕様やデータ構造

## ❓ 困ったときは

1. **インストールエラー**  
   → [QUICKSTART.md](./QUICKSTART.md) のトラブルシューティング

2. **Firebase接続エラー**

3. **アーキテクチャ理解**  
   → [ARCHITECTURE.md](./ARCHITECTURE.md) のデータフロー図
